# Docker Lumen App
PHP web app using [Docker](https://docker.com) and [Lumen Framework](https://lumen.laravel.com/).

# Quick Start
1. Clone this repo
2. Run `docker-compose up -d --force-recreate`

# Test the server is running
`curl localhost`
